#### 3.2. Exclude static files

Target > Scope > Use advanced scope control > Exclude from scope > File
```
.*\.css\b|.*\.js\b|.*\.png\b|.*\.gif\b|.*\.ico\b|.*\.jpeg\b|.*\.jpg\b|.*\.ttf\b|.*\.svg\b|.*\.woff2\b
```
